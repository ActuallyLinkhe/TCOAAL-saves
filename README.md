# TCOAAL Saves

Collection of save files for "The Coffin of Andy and Leyley"

Save files are stored in:\
`%appdata%\CoffinAndyLeyley` on windows\
`<SteamLibrary-folder>/steamapps/compatdata/2378900/pfx/` on Linux through Steam Play


Open the game first and leave it open, move the save file to your saves folder and then restart the game. You can also rename the save file to change the slot it will occupy, highest accepted being `file50.rpgsave`.

### About Unlocking Stars and Vision Room:
Both the stars and the Vision Room itself work similarly to achievents, as in, they're saved on a file separate from your usual saves.

In order to unlock the Vision Room, all you need is to reach chapter 3 on a save file where you saved the batteries in chapter 1. All saves heres tem from a playthrough where i saved the batteries, so any file will work.

If you just want the quickest way to unlock, load the [File 14](https://github.com/ActuallyLinkhe/TCOAAL-saves/blob/main/%5BCHAPTER%203%20Decay%5D%20Cliffhanger%20route%20(canon%20route)/file14.rpgsave) from chapter 3, interact with the toy kart, and after the dialogue you will have unlocked the Vision Room.

Stars work in a similar way, so just check the save files for each star in each chapter, load them onto your game, unlock the star and you'll be good to go.
